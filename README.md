# Getting Started

#### 1 - Clone the repository

```
git clone https://github.com/Bxstars/Player.git
cd Player
```
#### 2 - Install dependencies

```
npm install
```

#### 3 - Run the project

```
ng serve
```

#### Access in your browser:
```
http://localhost:4200
```
